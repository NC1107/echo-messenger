# Planned Features

Features under consideration for future releases, roughly ordered by priority.

## Near-term

- **iOS push notifications** -- APNs integration so iPhone users receive messages while the app is suspended
- **Account recovery** -- optional email on accounts + password reset flow
- **Global search** -- search messages across all conversations
- **Biometric login** -- Face ID / fingerprint unlock on mobile

## Mid-term

- **Threads** -- Slack-style threaded replies within a conversation
- **Polls** -- create and vote on polls in group chats
- **Device / session tracking** -- see active sessions and revoke them
- **Mobile screen sharing** -- share screen in voice lounges from phone
- **Cross-platform notifications** -- Android FCM, Web push API

## Long-term

- **Account switching** -- multi-account support without logging out
- **Improved auth** -- longer-lived tokens with anomaly detection, adaptive trust
- **Offline-first messaging** -- persistent message queue with automatic retry on reconnect
- **Group encryption** -- fully wired Signal-based group encryption (infrastructure exists)
